# gatsby + strapi + redux SET UP 

1. 사용 도구

* Frontend: Gatsby.js
* Backend: Strpi
* State management: Redux
* Graphql query 

2. 설명

- 정적 페이지 개발시 Gatsby 프로젝트에 Redux를 설정하여 Strapi database의 정보를 간단히 조회만 할 수 있는 페이지.
- Gatsby 특성상 CRUD 중 R의 기능만 사용 가능.
